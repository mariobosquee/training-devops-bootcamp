Cheat Sheet

Helm

Referencia rapida

01 — Repositorios

Repositorios

bash
helm repo add <nombre> <url> # Anadir un repositorio helm repo update # Actualizar indice local helm repo list # Listar repos configurados helm search repo <keyword> # Buscar charts en repos locales helm search hub <keyword> # Buscar en ArtifactHub helm repo remove <nombre> # Eliminar un repositorio

Ejemplo con Bitnami:

bash
helm repo add bitnami https://charts.bitnami.com/bitnami helm repo update helm search repo bitnami/nginx
02 — Instalar

Instalar

bash
helm install <release> <chart> # Instalacion basica helm install <release> <chart> -f values.yaml # Con fichero de values helm install <release> <chart> --set key=val # Sobreescribir un value helm install <release> <chart> -n <namespace> # En un namespace concreto helm install <release> <chart> --create-namespace # Crear namespace si no existe helm install <release> <chart> --dry-run # Simular sin instalar

Ejemplo:

bash
helm install mi-nginx bitnami/nginx \ -f values-custom.yaml \ --set service.type=ClusterIP \ -n web --create-namespace
03 — Gestionar releases

Gestionar releases

bash
helm list # Listar releases del namespace actual helm list -A # Listar releases de todos los namespaces helm status <release> # Estado detallado de un release helm history <release> # Historial de revisiones
bash
helm upgrade <release> <chart> # Actualizar un release helm upgrade <release> <chart> -f values.yaml # Actualizar con nuevos values helm upgrade --install <release> <chart> # Instalar si no existe, upgrade si existe
bash
helm rollback <release> <revision> # Volver a una revision anterior helm uninstall <release> # Eliminar un release helm uninstall <release> -n <ns> # Eliminar de un namespace concreto
04 — Inspeccionar charts

Inspeccionar charts

bash
helm show chart <chart> # Metadatos del chart (Chart.yaml) helm show values <chart> # Values por defecto del chart helm show readme <chart> # README del chart helm show all <chart> # Todo: chart + values + readme
bash
helm template <release> <chart> # Renderizar YAML sin instalar helm template <release> <chart> -f values.yaml # Renderizar con values custom helm get values <release> # Ver values de un release instalado helm get manifest <release> # Ver manifiestos de un release
05 — ArtifactHub

ArtifactHub

artifacthub.io es el buscador publico y centralizado de charts de Helm. Contiene miles de charts listos para usar — nginx, PostgreSQL, Redis, Grafana, ArgoCD, Prometheus y muchos mas. Cada chart incluye documentacion detallada de todos los values configurables, versiones disponibles y ejemplos de uso. Los charts mas populares estan mantenidos por la comunidad y por empresas como Bitnami. Antes de crear un chart propio, busca siempre aqui: lo mas probable es que ya exista uno listo para instalar.

bash
helm search hub nginx # Buscar "nginx" en ArtifactHub desde la CLI
06 — Flujo tipico

Flujo tipico

  1. Buscar el chart en ArtifactHub o con helm search
  2. Anadir el repositorio con helm repo add
  3. Inspeccionar los values por defecto con helm show values
  4. Crear un fichero values-custom.yaml con tu configuracion
  5. Instalar con helm install -f values-custom.yaml
  6. Verificar con helm status y kubectl get pods
  7. Actualizar con helm upgrade cuando cambien los values
  8. Si algo falla, helm rollback a la revision anterior
bash
# Flujo completo en 5 comandos helm repo add bitnami https://charts.bitnami.com/bitnami helm show values bitnami/nginx > values-custom.yaml # ... editar values-custom.yaml ... helm install mi-nginx bitnami/nginx -f values-custom.yaml -n web --create-namespace helm status mi-nginx -n web
