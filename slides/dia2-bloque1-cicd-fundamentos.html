El problema

El problema

Desplegar a mano es un desastre
  • Conectarse al servidor por SSH
  • Descargar codigo manualmente
  • Instalar dependencias
  • Parar y arrancar la aplicacion
  • Rezar para que funcione

Que es CI/CD

Que es CI/CD

CI/CD = Automatizar todo ese proceso

Tres conceptos, una escalera:

1. Continuous Integration (CI) -- Probar codigo automaticamente en cada cambio.

2. Continuous Delivery (CD) -- Empaquetar y dejar listo para desplegar.

3. Continuous Deployment (CD) -- Desplegar automaticamente sin intervencion.

Las etapas de un pipeline

Analogia -- La

Analogia -- La cadena de montaje

CI/CD es como una fabrica
  • Code = materia prima llega
  • Build = piezas se ensamblan
  • Test = control de calidad
  • Deploy = producto sale al mercado

Sin cadena de montaje: lento, caro, defectuoso.

Con cadena de montaje: rapido, repetible, fiable.
