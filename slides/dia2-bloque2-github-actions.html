Que es GitHub Action

Que es GitHub Actions

CI/CD integrado en GitHub
  • No hay que instalar nada.
  • No hay que crear cuentas externas.
  • Gratis para repositorios publicos.
  • Se configura con un archivo YAML en el repositorio.

Jerarquia de concept

Jerarquia de conceptos

Workflow > Job > Step 
Workflow  (el proceso completo)

|


+-- Job  (grupo de tareas en una maquina)


|


+-- Step  (una tarea individual)


|


+-- Action o comando

Un workflow tiene jobs.

Un job tiene steps.

Un step usa una action o ejecuta un comando.

Triggers -- Cua

Triggers -- Cuando se ejecuta

Eventos que disparan el workflow
  • `on: push` -- En cada push.
  • `on: push: branches: [main]` -- Solo en push a main.
  • `on: pull_request` -- En pull requests.
  • `on: workflow_dispatch` -- Manualmente.

Hoy usamos: push a main.

Secrets

Secrets

Variables seguras para credenciales
  • `GITHUB_TOKEN` -- Token automatico, ya incluido. No hay que configurar nada.
  • Secrets personalizados -- Settings > Secrets and variables > Actions.

Uso en el workflow:

password: ${{ secrets.GITHUB_TOKEN }}
